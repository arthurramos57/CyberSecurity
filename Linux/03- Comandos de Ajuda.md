Comandos de Ajuda

(ls): Comando utilizado para listar arquivos e diretórios dentro de uma pasta atual ou em qualquer diretório especificado.
(man nome do comando): Comando utilizado para saber quais opções de ferramentas o comando oferece, uma maneira de obter ajuda.
(nome do comando --help) : Comando utilizado para obter ajuda nas ferramentas que o comando oferece, sem precisar navegar pela documentação completa.
(apropos nome do comando): Comando pesquisa nas descrições instâncias de uma determinada palavra-chave no manual de ajuda.

Caso tenha dificuldades para entender um comando longo, você pode procurar ajuda nesse site: https://explainshell.com/
