Estrutura Linux

Linux é um sistema operacional usado em computadores pessoais, servidores e dispositivos moveis. Ele se destaca na CyberSecurity por sua robustez, flexibilidade 
e natureza de código aberto. Linux é um sistema operacional, assim como Windows, MacOS, IOS e o Android. Um sistema operacional (SO) um software que 
gerencia todos os recursos de hardware de um computador que facilita a comunicação entre aplicações de software e componentes do hardware. Diferente de 
alguns sistemas operacionais, o Linux vem de muitas distribuições diferentes que são chamada de "distrito" que são versões do Linux adaptadas a diversas 
necessidades e preferência.

Padrão de Hierarquia do sistema de arquivos

(/): Diretório de nível superior e é o arquivo raiz do sistema pois contém todo os arquivos necessário para inicialização. 
(/bin): Contém binários de comandos essencial. 
(/boot): bootlander estático, executável no kernel é o arquivo necessário para inicializar o sistema. 
(/dev): Contém arquivo e dispositivo para facilitar o acesso a todos os dispositivos de hardware conectado no sistema. 
(/etc): Arquivo de configuração do sistema local. (/home): Subdiretório de armazenamento de cada usuário. 
(/lib): Arquivo de biblioteca compartilhada que são necessário para inicializar o sistema. 
(/media): Media removível como drives USB. (/mnt): Montagem temporária para sistema de arquivos comuns. 
(/opt): Arquivo opcionais, ferramentas de terceiro, ser salvo aqui. 
(/root):D iretorio inicial de usuários root. 
(/sbin): Executável usado para administração do sistemas (arquivos binários do sistema). 
(/tmp): Sistema Operacional e muitos programas usam esse diretório para armazenar arquivos temporários. 
(/usr): Executáveis, bibliotecas, arquivos man, etc. 
(/rar): Diretório que contém arquivos de log, caixa de entrada de e-mail, arquivos relacionados a aplicação web, arquivos cron e mais.
